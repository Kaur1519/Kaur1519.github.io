Emphasis tag(em)
The tag is used to define slightly more important text.A screen reader will pronounce the text with an emphasis.
Abbrevation tag(abbr)
The abbr element is used if you use an abbreviation in one part of the page to define it and provide the full expansion using the title attribute.
Mark tag(mark)
The mark tag is used to highlight a piece of text reference.

The blockquote tag represents a section that is quoted from another source
